Що таке Bootstrap?

  • Bootstrap це безкоштовна бібліотека для прстого та швидкого створення сайтів.
  • Bootstrap охоплює більшість шаблонів дизайну HTML та CSS кнопки, зображення навігацію та плагіни JavaScript.
  • Bootstrap 4 повністю вільний продукт.

Чому використовують Bootstrap?

  • Простиий для використання: адже потрiбні тыльки базовы знання HTML та CSS.
  • Адаптивність: Bootstrap включає адаптивний CSS для підлаштування під розширення телефонів,планшетів та повноцінних комп*ютерів
  • Розповсюдженість: Bootstrap підтримується всіма сучасними браузерами.

Історія Bootstrap.

  • Авторами Bootstraр є Марк Отто та Джейкоб Торнтон,open source версія була завантажена на GihHub у серпні 2011.
  • В червні 2014 Bootstrap став продуктом №1 на Github.

Де дістати Bootstrap?

Завантажити host Bootstrap можна на сайті getbootstrap.com, на ньому ж же знайдете офіційну документацію

Поширення засобами CDN

Якщо не хочете завантажувати host Bootstrap Ви зажди можете використати CDN дистрибуцію.

MaxCDN:

<!-- Latest compiled and minified CSS -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.0/css/bootstrap.min.css">

<!-- jQuery library -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>

<!-- Latest compiled JavaScript -->
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.0/js/bootstrap.min.js"></script>

Kілька слів за jQuery

Bootstrap викоритстовує бібліотеку jQuery для JavaScript доповнень (моделей, барів, анімацій і т.д.) Якщо Ви використовуєте тільки CSS частину Bootstrap підключати jQuery не обов'язково.

Створіть свою першу сторінку використовуючи Bootstrap.

Bootstrap використовує HTML та CSS елементи, які вимагають HTML5 стандарт.

1.Створіть HTML шаблон.

Завжди використовуйе HTML5 стандарт разом з атирибутом "lang" при створенні нової сторінки.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
  </head>
</html>

2. Адаптивний дизайн

Bootstrap 3 проектується, щоб бути адаптивним до мобільних пристроїв. Стандарті розширення екраів вже додані до бібліотеки.
Для забезпечення нележної роботи потірбно додани наступний тег до змісту "head"

<meta name="viewport" content="width=device-width, initial-scale=1">

width=device-width це частина налаштувань розміру сторінки до розширення екрану пристрою.

initial-scale=1 встановлює початковий зуи при завантажуванні сторінки.

3. Контейнери

Bootstrap 4 обовязково вимагає роміщенні обєктів сайту в контейнерах.
Існує два види контейнерів

  • Адаптивний .container
  • Незмінний .container-fluid, який займає весь простір при перегляді.
.container
.container-fluid

Дві основні сторінки Bootstrap.

Наступний приклад показує код для основної сторінки Bootstrap (з адаптивним фіксованим width контейнером):

<!DOCTYPE html>

</head>

  <h1>My First Bootstrap Page</h1>

</body>
</html>

Система таблиць в Bootstrap 4

Система заснована на контейнерах які можуть бути розміщені в 12 колоноках.
Якщо ви не хочете користуватися усіма 12 колоноками індивідуально, ви можете групувати колонки разом, щоб створити ширші колонки:

span 1 span 1 span 1 span 1 span 1 span 1 span 1 span 1 span 1 span 1 span 1 span 1
 span 4  span 4  span 4
span 4 span 8
span 6 span 6
span 12

Дана система є адаптивною і автоматично змінює відображення залежно від розміру екрану
Переконайтеся, що в сумі використовуєте12 або менше (це не потрібно, щоб ви користувалися усіма 12 доступними колоноками).

Класифікація колонок:

  • .col- (дуже малі пристрої - з розширенням менше 576px)
  • .col-sm- (малі пристрої - з розширенням рівним або більшим 576px)
  • .col-md- (середні пристрої - з розширенням рівним або більшим 768px)
  • .col-lg- (великі пристрої - з розширенням рівним або більшим 992px))
  • .col-xl- (дуже великі пристрої - з розширенням рівним або більшим 1200px)

Дані класи можна комбінувати, щоб створити більше динамічних і гнучких розташувань

Увага! Якщо ви бажаєте встановити ту ж ширину для sm і md, вам треба конкретизувати тільки sm.

Базова структура таблиці Bootstrap.

Приклад коду сторінки з таблицею

<!-- Control the column width, and how they should appear on different devices -->
<div class="row">
  <div class="col-*-*"></div>
  <div class="col-*-*"></div>
</div>
<div class="row">
  <div class="col-*-*"></div>
  <div class="col-*-*"></div>
  <div class="col-*-*"></div>
</div>

<!-- Or let Bootstrap automatically handle the layout -->
<div class="row">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>

Перший приклад: створюємо рядок div class="row", тоді додайтємо бажане число колонок (вказуємо з тим, що відповідає потрібному .col-*-* класу). Перша зірка (*) вказує тип: sm, md, lg або xl, а друга зірка вказує номер, який повинен позначити ширину для кожного ряду.


Другий приклад: замість вказування кількості колонок можна вказати розмір у %

Нижче ми зібрали деякі приклади основного Bootstrap розташування.

Текст та його властивості в Bootstrap 4

За промовчуванням Bootstrap 4 використовує розмір шрифту 16px, та інтервал 1.5.

Стандартні шрифти сімї "Helvetica Neue" - Helvetica, Arial, sans-serif.

Окрім того всі

елементи мають верхній відступ: 0 та нижній відступ 16px.

h1-h6

Bootstrap 4 стилі HTML заголовків (h1 - h6) відрізняються своїми розмірами від стандартних.

h1 Bootstrap heading (2.5rem = 40px)

h2 Bootstrap heading (2rem = 32px)

h3 Bootstrap heading (1.75rem = 28px)

h4 Bootstrap heading (1.5rem = 24px)

h5 Bootstrap heading (1.25rem = 20px)
h6 Bootstrap heading (1rem = 16px)

Відображення заголовків

Відображення заголовків використовує 4 маштаби екранів, залежно від розширення

Display 1

Display 2

Display 3

Display 4

/small/

В Bootstrap 4 є /small/ елемент для відображення, вторинного тексту:

h1 heading secondary text

h2 heading secondary text

h3 heading secondary text

h4 heading secondary text

h5 heading secondary text
h6 heading secondary text

/mark/

В Bootstrap 4 є /mark/ елемент з жовтим кольором фону і деяким заповненням:

Use the mark element to highlight text.

/abbr/

В Bootstrap 4 є /abbr/ eлемент з пунктирним низом межі:

The WHO was founded in 1948.

/blockquote/

Додайте .blockquote клас /blockquote/ вказуючи джерело вмісту:

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.


From WWF's website
